This is a [Next.js](https://nextjs.org/) project, with typescript

## Getting Started

First, run the development server:

```bash
yarn dev
```

## Functionalities

- uses the [API cocktaildb](https://www.thecocktaildb.com/api.php)
- offer a random cocktail suggestions
- can search cocktails by name or ingredients
- can create an account
- if logged in, can save cocktails as favorite
